================================================================================
COMPARISON WITH ORIGINAL PIXOO LIBRARY
================================================================================

üìö Original pixoo workflow:
   1. pixoo = Pixoo(ip_address)
   2. pixoo.fill(Palette.BLACK)  # Clear buffer
   3. pixoo.draw_pixel((x, y), Palette.RED)
   4. pixoo.draw_line((x1, y1), (x2, y2), Palette.GREEN)
   5. pixoo.draw_rectangle((x1, y1), (x2, y2), Palette.BLUE)
   6. pixoo.draw_text('Hello', (x, y), Palette.WHITE)
   7. pixoo.draw_image('path.png', (x, y))
   8. pixoo.push()  # Send buffer to device

üîÑ PixooAsync current workflow:
   1. async with PixooAsync(ip_address) as pixoo:
   2.     await pixoo.initialize()  # Prepare buffer
   3.     # Drawing primitives not yet in public API
   4.     await pixoo.send_text(text, xy, color, ...)
   5.     await pixoo.push()  # Send buffer to device

‚ú® What needs to be added:
   - Public methods for all drawing primitives
   - Buffer manipulation (fill, clear)
   - Image drawing from PIL objects
   - Character/text rendering at pixel coordinates
